#
SmartSearch - LLM Powered AI Chatbot

SmartSearch is a web-based AI chatbot application built using ASP.NET Core and powered by a Large Language Model (LLM). The system enables users to ask questions and receive intelligent responses in real time through a simple and interactive web interface.

Overview

The project demonstrates how Large Language Models can be integrated with a modern web application to provide conversational AI capabilities. It processes user queries, sends them to an LLM API, and returns meaningful responses to the user.

Features

AI powered conversational chatbot
Integration with Large Language Models (LLM)
Real time user interaction
Clean and responsive interface
Scalable backend using ASP.NET Core
Easy integration with external AI APIs

Installation

Clone the repository

git clone https://github.com/this-is-mansi/ChatSearch.git

Navigate to the project directory

cd AIChatBot

Run the application

dotnet run

Usage

Open the application in a web browser.
Enter a message in the chatbot interface.
The request is sent to the LLM API and a response is generated and displayed.
